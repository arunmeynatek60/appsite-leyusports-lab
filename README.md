# appsite-leyusports-lab

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定域名或网站，而是作为通用 HTML 资源的集合，便于版本管理和在线预览。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立页面，命名清晰，便于识别。
- `assets/`：存放页面间共享的样式、脚本或静态资源。

## 页面归档说明

- 所有页面均为纯 HTML 实现，不依赖后端服务。
- 页面之间相互独立，无耦合关系。
- 每个页面可单独通过 GitHub Pages 或其他静态托管服务访问。
- 归档内容可能包含演示页、工具页、实验性页面等，不包含任何商业或推广性质内容。

## 使用方式

1. 克隆本仓库：
   ```bash
   git clone https://github.com/your-username/appsite-leyusports-lab.git
   ```
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
3. 如需部署，可将 `pages/` 目录或整个仓库配置为静态站点根目录。
